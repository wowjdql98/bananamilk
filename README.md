# bananamilk 반응형 웹 홈페이지

## 개요
  가변 Full Screen Slide 유형의 홈페이지로 바나나맛우유 홈페이지를 만들었습니다.

## 작업내용
0. 시안 및 프로토타입 제작(figma)
1. 헤더 toggle gnb 반응형으로 구현
2. main visual 영역 애니메이션(fade-in)
3. media query와 flex layout 활용
4. 동영상 모달창 구현
5. smooth page scroll


## 라이브러리
- 제이쿼리
- AOS
- Bootstrap 아이콘
